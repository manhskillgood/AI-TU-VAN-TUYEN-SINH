## 🚀 Cách Chạy Ứng Dụng

### Bước 1: Chuẩn Bị (1 phút)
```bash
cd c:\Users\Manh\DO_AN_TN
flutter pub get
```

### Bước 2: Cấu Hình Firebase (5 phút)
1. Tạo Firebase project tại https://firebase.google.com/
2. Download google-services.json
3. Copy vào android/app/
4. Sửa lib/firebase_options.dart

### Bước 3: Cấu Hình Google AI (2 phút)
1. Lấy API key từ https://ai.google.dev/
2. Thêm vào lib/services/ai_service.dart

### Bước 4: Chạy (1 phút)
```bash
flutter run              # Android
flutter run -d iPhone    # iOS
flutter run -d chrome    # Web
```
