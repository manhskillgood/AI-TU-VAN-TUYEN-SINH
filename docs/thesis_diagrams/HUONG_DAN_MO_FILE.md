# Hướng dẫn mở sơ đồ draw.io cho báo cáo

## Cách 1 — draw.io trên web (khuyên dùng)

1. Mở https://app.diagrams.net/
2. **File → Open from → Device**
3. Chọn file `.drawio` trong thư mục này
4. Chỉnh chữ / kéo lại cho đẹp (nếu cần)
5. **File → Export as → PNG** (Zoom 200%, Border 10)
6. Chèn PNG vào Word, caption: *Hình 2.x...*

## Cách 2 — VS Code / Cursor

1. Cài extension **Draw.io Integration**
2. Click đúp file `.drawio` để mở và sửa

## Danh sách file

| File | Nội dung |
|------|----------|
| `Hinh_2_1_Use_Case.drawio` | Use Case tổng quát |
| `Hinh_2_2_ER.drawio` | Quan hệ dữ liệu logic |
| `Hinh_2_3_Kien_truc.drawio` | Kiến trúc 3 tầng |
| `Hinh_2_4_Sequence.drawio` | Sequence gợi ý ngành |
| `Hinh_2_5_Screen_Flow.drawio` | Điều hướng màn hình |
| `Hinh_3_3_Wizard_Nhap_Lieu.drawio` | Các bước nhập liệu wizard định hướng (Chương 3) |

## Lưu ý khi nộp

- Xuất PNG **nền trắng**, độ phân giải cao
- Trong Word: căn giữa hình, caption phía dưới (Italic)
- Cập nhật **Danh mục hình** ở đầu báo cáo
